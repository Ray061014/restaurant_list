# Restaurant_List

你喜歡和家人朋友一起談天說笑但卻找不到合適的地點嗎？這個餐廳網站集合了美式、義式、日式等不同的異國美食，讓你可以盡情享受和家人朋友相聚的時光。

## Installing

1. 下載此專案到本機端
 ```
   git clone https://github.com/Ray061014/restaurant_list.git
 ```
2. 進入此專案資料夾
3. 輸入`npm install` ，下載此專案所需的套件
 ```
   npm install
 ```
 4. 執行此專案
 ```
   npm run dev
 ```
 
 ## Features
 + 索引頁可察看所有餐廳(名字、種類、評分)
 + 搜尋框中可尋找你喜歡的餐廳或料理種類
 + 點擊你有興趣的餐廳圖片，即可察看更豐富的資料(類別、地址、電話、詳細說明及圖片)
